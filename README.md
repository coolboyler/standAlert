# 站立冒险 - StandUp Adventure 🚶‍♂️✨

一个功能丰富、创意十足的站立提醒网站，将久坐提醒变成一场有趣的冒险旅程！

## 🎯 项目特色

### 🐾 虚拟宠物系统
- **专属宠物伙伴**：Standy 会陪伴你的每一次站立
- **情感互动**：宠物有幸福度、饥饿度、能量值等状态
- **成长系统**：通过站立获得经验，宠物会升级进化
- **日志记录**：记录与宠物的每一个互动瞬间

### 🏆 成就系统
- **10+ 独特成就**：从新手到达人，应有尽有
- **分类追踪**：连胜、里程碑、时间特殊等分类
- **进度可视化**：清晰的完成度展示
- **奖励机制**：解锁成就获得积分奖励

### 🎮 创意小游戏
- **平衡挑战**：单腿站立测试稳定性
- **眼球追踪**：放松眼部肌肉的趣味游戏
- **呼吸节奏**：配合呼吸的放松练习
- **反应测试**：快速点击挑战反应速度
- **记忆配对**：锻炼记忆力的卡片游戏
- **趣味问答**：健康知识小测试

### 🎵 环境音效
- **自然音景**：森林、海洋、雨声、咖啡馆等
- **提醒音效**：多种提醒声音可选
- **音量控制**：精细的音量调节
- **可视化反馈**：音效播放时的视觉提示

### 📊 数据统计
- **健康影响估算**：卡路里消耗、站立时长
- **历史图表**：周度站立数据可视化
- **详细统计**：连胜、平均值、总时长等
- **数据导出**：备份和恢复功能

### ⚙️ 智能设置
- **自定义提醒**：30分钟到2小时可调
- **通知系统**：浏览器通知 + 视觉 + 振动
- **主题切换**：5种精美主题场景
- **自动喂养**：站立时自动恢复宠物状态

## 📋 核心功能详解

### 1. 智能提醒系统
- 可配置的提醒间隔（30分钟 - 2小时）
- 多模态提醒：视觉、听觉、触觉
- 智能调度：自动计算下次提醒时间
- 稍后提醒：5分钟延时功能

### 2. 虚拟宠物系统
```javascript
// 宠物状态示例
{
  name: "Standy",
  happiness: 100,  // 幸福度
  hunger: 100,     // 饥饿度
  energy: 100,     // 能量值
  level: 1,        // 等级
  xp: 0            // 经验值
}
```

### 3. 成就解锁机制
- **首次站立**：完成第一次站立
- **连胜系列**：3天、7天、30天连胜
- **里程碑**：100次、1000次站立
- **时间特殊**：早起鸟、夜猫子、周末战士

### 4. 健康数据追踪
- **卡路里估算**：每次站立约消耗25卡路里
- **站立时长**：累计站立时间统计
- **健康评分**：综合健康指标
- **趋势分析**：周度数据图表

## 🎮 游戏玩法

### 平衡挑战
1. 点击开始按钮
2. 抬起一条腿保持平衡
3. 坚持30秒获得高分
4. 保持时间越长，得分越高

### 眼球追踪
1. 用眼睛跟随红点移动
2. 点击红点获得分数
3. 持续20秒
4. 锻炼眼部肌肉

### 呼吸练习
1. 跟随提示进行呼吸
2. 吸气4秒，屏住4秒，呼气6秒
3. 完成30秒练习
4. 放松身心

## 📊 数据管理

### 导出数据
- 下载JSON格式的完整数据备份
- 包含所有统计数据和设置
- 可用于数据迁移或分析

### 导入数据
- 恢复之前备份的数据
- 自动合并冲突
- 保持数据完整性

### 数据清除
- 一键重置所有数据
- 需要双重确认
- 不可恢复操作

## 🔒 隐私保护

- **本地存储**：所有数据存储在本地浏览器
- **无网络请求**：不发送任何数据到服务器
- **数据导出**：用户完全控制数据备份
- **无追踪**：无分析或用户行为追踪
