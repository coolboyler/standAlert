# 站立冒险项目 - 完整设计总结 🚶‍♂️✨

## 🎯 项目概述

**站立冒险**是一个将久坐提醒转化为有趣冒险体验的创意网站。通过游戏化、虚拟宠物和丰富的互动功能，让健康提醒变得生动有趣。

## 🏗️ 项目架构

### 核心技术栈
- **前端框架**: Vue 3 (Composition API)
- **状态管理**: Pinia
- **路由管理**: Vue Router
- **构建工具**: Vite
- **样式预处理**: Sass/SCSS
- **工具库**: Day.js (时间处理)

### 项目结构
```
stand-alert/
├── src/
│   ├── components/          # 可复用组件
│   ├── stores/             # Pinia状态管理
│   │   ├── user.js         # 用户数据和业务逻辑
│   │   └── audio.js        # 音效管理系统

│   ├── utils/              # 工具函数

│   ├── router/             # 路由配置
│   └── main.js             # 应用入口
├── public/                 # 静态资源
└── 配置文件
```

## 🔧 核心算法

### 1. 连胜计算
```javascript
function updateStreak() {
  const today = new Date().toDateString()
  const lastStand = getLastStandDate()

  if (lastStand === today) return        // 今天已站立

  const yesterday = getYesterdayDate()
  if (lastStand === yesterday) {
    currentStreak++                      // 连续两天
  } else if (lastStand !== today) {
    currentStreak = 1                    // 断签重置
  }
}
```

### 2. 成就检查
```javascript
function checkAchievements() {
  const checks = {
    'first_stand': () => totalStands >= 1,
    'streak_3': () => currentStreak >= 3,
    'century': () => totalStands >= 100,
    'early_bird': () => {
      const hour = new Date().getHours()
      return hour >= 5 && hour <= 7
    }
  }

  achievements.forEach(ach => {
    if (!ach.unlocked && checks[ach.id]()) {
      unlockAchievement(ach)
    }
  })
}
```

### 3. 宠物状态衰减
```javascript
setInterval(() => {
  pet.happiness = Math.max(0, pet.happiness - 0.1)
  pet.hunger = Math.max(0, pet.hunger - 0.05)
  pet.energy = Math.max(0, pet.energy - 0.08)
}, 60000) // 每分钟
```

## 🎮 游戏机制详解

### 平衡挑战
- **目标**: 单腿站立保持平衡
- **计时**: 0-30秒
- **得分**: 每秒10分，满分300
- **完成**: 坚持5秒以上

### 眼球追踪
- **目标**: 点击随机移动的红点
- **频率**: 1.5秒移动一次
- **得分**: 每次点击+5分
- **时长**: 20秒

### 呼吸练习
- **节奏**: 吸气4秒 → 屏住4秒 → 呼气6秒
- **循环**: 重复进行
- **奖励**: 完成30秒+150分

### 反应测试
- **机制**: 随机出现按钮，快速点击
- **计时**: 15秒
- **得分**: 根据反应时间计算
- **记录**: 最快反应时间

### 记忆配对
- **规则**: 4x4卡片，8对图案
- **限制**: 无时间限制
- **得分**: 步数越少分越高
- **完成**: 所有配对完成

### 趣味问答
- **题目**: 4道健康知识题
- **机制**: 选择正确答案
- **得分**: 每题25分
- **反馈**: 即时对错提示
