## 🚀 快速启动

### 方式一：本地开发
```bash
# 1. 进入项目目录
cd stand-alert

# 2. 安装依赖
npm install

# 3. 启动开发服务器
npm run dev

# 4. 打开浏览器访问
# http://localhost:3000
```

### 方式二：查看概念演示
```bash
# 直接在浏览器打开 demo.html
open demo.html
```

### 方式三：构建生产版本
```bash
# 构建优化后的生产版本
npm run build

# 预览构建结果
npm run preview
```
